This is my Print Project. For this project, we were assigned to make a bookmark for elementary aged students. We started out by hand sketching 5 different ideas, from which we chose our favorite. We then made the chosen bookmark in InDesign. After that, we got into groups and put our bookmarks into a single file so they can be printed out together.

This was the first project we were assigned, so it served as an introduction to InDesign and how to make ideas into reality.
